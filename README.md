# 📖 About

**JobPortalWeb** is a complete recruitment platform built using **Python Django**. It enables job seekers to search and apply for jobs while employers can post vacancies, manage applicants, and streamline the hiring process.

This project follows the MVC architecture provided by Django and includes responsive UI, authentication, job management, and admin functionalities.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/dhanakodi7481/Jobporalweb.git
```

## Navigate

```bash
cd Jobportalweb
```

## Create Virtual Environment

```bash
python -m venv venv
```

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

---

## Install Requirements

```bash
pip install -r requirements.txt
```

---

## Apply Migrations

```bash
python manage.py migrate
```

---

## Run Server

```bash
python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```
